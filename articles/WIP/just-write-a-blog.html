为什么写博客

I want to write a blog, that's it.

为什么要写博客? 我想不同的人可能有不同的想法, 对于我来说, 我写博客是为了表达我的思想, 记录我的生活.

记得在大学的时候就开始折腾博客, 和室友小林一起买服务器备案, 阿里云备案还要在一个很丑的蓝色幕布上拍照:), 刚开始折腾动态博客, 安装 wordpress,搞 LNMP 建站, 还有 360 的 firekylin, 后来又去搞 github pages 的静态博客. 哦对了, 还有这 CSDN 博客园也写了一点博客. 折腾了这么多, 博客却没写几篇.

没能坚持下来我想有几个原因，一是不知道写什么，写得太简单，之前有很多博客都是一些学习笔记，而我又觉得网上能查到的东西不需要记笔记，这是另外一个话题了，后面可能会写文章讨论一下是否需要做笔记，二是有时候觉得自己写的东西逻辑不够通顺，文字功底不行，三是有时候不想把自己内心的一些想法表达出来让别人看到.

但现在我的想法又有些不同了，写什么？我觉得只要是自己的想法都值得被记录下来，不管是对某些事物的看法，或者是自己某时某刻的心情，或者是一些完整的成体系化的一些知识，把博客当成自己的生活记录，多年之后再回过头看看。文字功底不行，这个需要多写，慢慢练习，这也是我在其他人的博客里看到的: 写作12年，我的经验和技巧，同样，你写的博客在某个不经意的时刻也可能会对别人产生影响.至于表达观点或者心情，不想让别人看到，我现在想的就是不要在意别人的看法，生活是自己的，为什么要在别人的眼光里活着. 还有一点就是写作时间太长，但是没关系，一篇文章你可以写几天，几个星期，几个月，甚至几年.只要一直在写就够了.
